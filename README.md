# 코딩 자율학습 FastAPI 파이썬 백엔드 개발 입문

이 저장소에는 <코딩 자율학습 FastAPI 파이썬 백엔드 개발 입문>의 소스코드가 담겨 있습니다.

/표지 업데이트 예정/

## 소스코드 다운로드 및 확인 방법

- 화면 위쪽에 보이는 [Code] 버튼을 클릭합니다.
- 아래로 펼쳐지는 메뉴에서 [Download ZIP]을 클릭해 압축 파일을 내려받습니다.
- 내려받은 파일의 압축을 풀면 3개의 폴더가 있습니다.
  - todos: 본문의 Todo(할 일 관리) 애플리케이션 코드입니다. 3~7장까지 모두 실습했을 때 완성 코드가 들어 있습니다.
  - blog: 셀프체크를 모두 실습했을 때 완성 코드입니다.
  - source: 해당 장이 끝났을 때를 기준으로 정리한 코드입니다. 이 폴더를 불러오면 중간 장부터 실습할 수 있습니다(예: 5장부터 실습하려면 todos4를 불러와 사용).
